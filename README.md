## ğŸ“¦ Installation

### Prerequisites

- Node.js 20+
- npm

### Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/blacksof-dev/mirach.git
   cd mirach
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run the development server**

   ```bash
   npm run dev
   ```

4. **Open your browser**

   Navigate to [http://localhost:3000](http://localhost:3000) to see the application.
